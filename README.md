Installation
To install and run this project locally, please follow these steps:

<h6>Clone the repository: Start by cloning this repository to your local machine using the following command:</h6>
<li>git clone https://github.com/anandpiyush1521/SnapBlog.git</li>

<h6>Install dependencies: Navigate to the project's root directory and install the necessary dependencies by running the following command:</h6>
<li>npm install</li>

<h6>Run the application: Once the dependencies are installed and the project is properly configured, start the application by running the following command:</h6>
<li>npm start</li>

This will start the Node.js server and make the application available locally.
